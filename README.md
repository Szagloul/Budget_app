# Budget App CLI

A command-line budgeting application built in Python. This app allows users to create spending categories, track deposits and withdrawals, transfer funds between categories, and visualize spending with a text-based bar chart.

## Features


- Category Management
  - Create multiple spending categories (e.g., Food, Entertainment, Clothing)  
  - Track deposits, withdrawals, and transfers per category  
- Ledger System
  - Record transactions with descriptions  
  - Print detailed category ledgers showing amounts and balances  
- Transfers Between Categories
  - Move money from one category to another  
  - Automatically update ledgers for both source and destination  
- Spending Visualization
  - Generate a bar chart showing percentage spent by category  
  - Percentages calculated from withdrawals only  
  - Category names displayed vertically under the chart  
- Interactive Command-Line Interface
  - Navigate menus to add categories, deposit, withdraw, transfer, check balances, and show ledgers  
  - Quit program when finished 

## Usage

1. Clone or download the repository.  
2. Open a terminal or command prompt.  
3. Run the program with Python 3.x:
```bash
python budget_app.py
